MochiNav V5.5.0

  • Connection:
    1. Added a feature to adjust the auto-off time for BLE when there is no connection with the phone: 5, 10, 15, 20, 25, or 30 minutes.
  • Expressions:
    1. Retains 84 expressions instead of the official commercial version‚Äôs 102 expressions.
    2. Added several options for emoji playback: Random - Sequential, Delay - No delay, Wink, etc.
  • Sound:
    1. Added a master ON/OFF switch for all device sounds, and separate ON/OFF for emoji sounds.
    2. The ESP32-C3 Super Mini has only 4MB of flash memory, so only 102 expressions + 2 intro sounds are stored internally. Emoji sounds will be stored on an SD card.
    3. SD card must be formatted in FAT32. Emoji sounds are in Wav 16-bit 22050Hz Mono format, numbered from 1 to 102.
    4. In this version, full emoji sounds are not yet released, so expressions still have no sound.
  • Added Update Menu (In Development):
    1. Allows entering and saving WiFi SSID and Password into the device.
    2. Displays the current version and checks for updates when a new version is available.
    3. Added language switcher: Currently supports two main languages, Vietnamese and English, adjustable directly in the Chronos app or in Menu -> Update.
  • Clock:
    1. Added lunar calendar display in the clock interface.
    2. Added weather forecast and current temperature synced from the Chronos app.
  • Performance Optimization:
    The program has been optimized for stability and faster performance.
  • Quick Shortcuts: Added quick shortcuts in the emoji interface:
    1. If a button is present, press once or press "Find Watch" to switch to the Clock screen.
    2. Press the button twice quickly to toggle random or sequential animations.
    3. Press the button three times quickly to toggle sound.
    4. Press the button four times quickly to toggle the delay time when playing random animations.
  • Bug Fixes:
    Fixed some issues such as settings not being saved, etc.
  • More to come: .....

Copy the "audio.bin" file downloaded in Step 1 directly to the memory card (do not create a folder). The file name must be "audio.bin"
